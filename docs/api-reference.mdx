This page documents the SDK surface area for all supported languages.

## JavaScript SDK

```js
import {
  MightyClient,
  MightyWebDevice,
  DEFAULT_BASE_URLS,
  decodeRawToRgb,
} from "./js/sdk/index.js";
```

### Exports

- `MightyClient`
- `MightyWebDevice`
- `DEFAULT_BASE_URLS: string[]`
- `decodeRawToRgb(frame, rawFormat?)`

### `MightyWebDevice`

Constructor:

```ts
new MightyWebDevice(options?: {
  baseUrl?: string;
  baseUrls?: string[] | null;
  streamPath?: string;      // default "/stream"
  commandPath?: string;     // default "/command"
  fetchImpl?: typeof fetch | null;
  headers?: Record<string, string> | null;
})
```

Methods:

- `getInfo(): { transport: "http"; source: string }`
- `connect(onBytes: (chunk: Uint8Array) => void): Promise<void>`
- `disconnect(): Promise<void>`
- `sendCommandPayload(cmdPayload: Uint8Array): Promise<Uint8Array>`

### `MightyClient`

Constructor:

```ts
new MightyClient(
  device: {
    connect(onBytes: (chunk: Uint8Array) => Promise<void> | void): Promise<void> | void;
    disconnect(): Promise<void> | void;
    sendCommandPayload?(payload: Uint8Array): Promise<Uint8Array>;
  },
  opts?: {
    commandTimeoutMs?: number;        // default 2000
    autoReconnect?: boolean;          // default true
    reconnectDelayMs?: number;        // default 300
    streamStallTimeoutMs?: number;    // default 3000; <=0 disables
    emitStatAsStatus?: boolean;       // default true
    normalizeChannelAliases?: boolean;// default true
  }
)
```

Lifecycle and stats:

- `connect(): Promise<void>`
- `disconnect(): Promise<void>`
- `isConnected(): boolean`
- `stats(): { rxFrames: number; rxBytes: number; decodeErrors: number; reconnects: number; commandTimeouts: number }`

Command/config:

- `command(name: string, data?: Uint8Array): Promise<{ ok: boolean; reqId: number; status: number; message: string; data: Uint8Array }>`
- `configGet(key: string, opts?: { as?: "text" }): Promise<{ ok: boolean; found: boolean; key: string; value: Uint8Array | string; message: string }>`
- `configSet(key: string, value: Uint8Array | string): Promise<{ ok: boolean; key: string; value: Uint8Array; message: string }>`
- `startVio(): Promise<{ ok: boolean; reqId: number; status: number; message: string; data: Uint8Array }>`
- `stopVio(): Promise<{ ok: boolean; reqId: number; status: number; message: string; data: Uint8Array }>`

Subscriptions (each returns `() => void` to unsubscribe):

- `onImage(cb)`
- `onPose(cb)`
- `onImu(cb)`
- `onVioState(cb)`
- `onViz(cb)`
- `onStatus(cb)`
- `onReset(cb)`
- `onAny(cb)`
- `onError(cb)`

### JavaScript callback payloads

`onImage`:

```ts
type RawImageFrame = {
  kind: "raw";
  timestampNs: bigint;
  width: number;
  height: number;
  format: number;
  channel: string;
  channelAlias?: "cam0" | "cam1";
  data: Uint8Array;
};

type StereoRawImageFrame = {
  kind: "stereo_raw";
  left: RawImageFrame;
  right: RawImageFrame;
};
```

`onPose`:

```ts
type PoseEvent = {
  poseType: "body" | "camera" | "other";
  poseTypeRaw: number;
  poseFlags: number;
  frameId: "odom";
  childFrameId: "base_link";
  positionM: [number, number, number];
  orientationXyzw?: [number, number, number, number];
  confidence: number;                   // [0, 1]
  isKeyframe: boolean;
  linearVelocityBodyMps?: [number, number, number];
  angularVelocityBodyRps?: [number, number, number];
  linearAccelerationBodyMps2?: [number, number, number];
  angularAccelerationBodyRps2?: [number, number, number];
  timestampNs?: bigint;
};
```

`onImu`:

```ts
type ImuEvent = {
  samples: Array<{
    timestampNs: bigint;
    ax: number;
    ay: number;
    az: number;
    gx: number;
    gy: number;
    gz: number;
  }>;
};
```

`onVioState`:

```ts
type VioStateEvent = {
  version: number;
  state: number;
  flags: number;
  timestampNs: bigint;
  fpsCurrent: number;
  fpsAverage: number;
  poseConfidence: number;
  trackingRate: number;
  numFeatures: number;
  loopClosures: number;
  buildVersion?: string;
  imuHzCurrent?: number;
  imuHzAverage5s?: number;
  initReasonCode: number;
  staticInitReasonCode: number;
  dynamicInitReasonCode: number;
  memoryTotalBytes?: bigint;
  memoryUsedBytes?: bigint;
  memoryFreeBytes?: bigint;
  lightLevel01?: number;
  lightRequired01?: number;
};
```

`onViz`:

```ts
type VizEvent =
  | { subtype: "features"; features: Array<{ x: number; y: number; status: number; id: number }>; rawPayload: Uint8Array }
  | { subtype: "detections"; detections: Array<{ x1: number; y1: number; x2: number; y2: number; label: string }>; rawPayload: Uint8Array }
  | { subtype: "matches"; matches: Array<{ x1: number; y1: number; x2: number; y2: number; confidence: number }>; rawPayload: Uint8Array }
  | { subtype: "unknown"; rawSubtype: number; rawPayload: Uint8Array };
```

`onStatus`:

```ts
type StatusEvent = { text: string };
```

`onReset`:

```ts
type ResetEvent = { receivedAtMs: number };
```

`onAny`:

```ts
type AnyEvent =
  | { type: "image"; data: RawImageFrame | StereoRawImageFrame }
  | { type: "pose"; data: PoseEvent }
  | { type: "imu"; data: ImuEvent }
  | { type: "vio_state"; data: VioStateEvent }
  | { type: "viz"; data: VizEvent }
  | { type: "status"; data: StatusEvent }
  | { type: "reset"; data: ResetEvent }
  | { type: "unknown"; rawType: string; payload: Uint8Array };
```

`onError`:

```ts
type ErrorEvent = {
  scope: "transport" | "protocol" | "command" | "config";
  code: string;
  message: string;
  cause?: unknown;
};
```

## Python SDK

```python
from mighty_sdk import MightyClient, MightyWebDevice
```

### `MightyWebDevice`

Constructor:

```python
MightyWebDevice(
    base_url: str | None = None,
    base_urls: list[str] | None = None,
    stream_path: str = "/stream",
    command_path: str = "/command",
    headers: dict[str, str] | None = None,
    connect_timeout_s: float = 5.0,
    read_timeout_s: float = 1.0,
    read_chunk_size: int = 64 * 1024,
)
```

Methods:

- `get_info() -> dict[str, str]` with keys: `transport`, `source`
- `connect(on_bytes: Callable[[bytes], None]) -> None` (blocking stream loop)
- `disconnect() -> None`
- `send_command_payload(cmd_payload: bytes) -> bytes`

### `MightyClient`

Constructor:

```python
MightyClient(
    device,
    command_timeout_s: float = 2.0,
    auto_reconnect: bool = True,
    reconnect_delay_s: float = 0.3,
    emit_stat_as_status: bool = True,
    normalize_channel_aliases: bool = True,
)
```

Lifecycle and stats:

- `connect() -> None` (starts background thread)
- `disconnect() -> None`
- `is_connected() -> bool`
- `stats() -> dict[str, int]` with keys:
  - `rx_frames`, `rx_bytes`, `decode_errors`, `reconnects`, `command_timeouts`

Command/config:

- `command(name: str, data: bytes = b"") -> dict[str, Any]`
  - keys: `ok: bool`, `req_id: int`, `status: int`, `message: str`, `data: bytes`
- `config_get(key: str, as_text: bool = False) -> dict[str, Any]`
  - keys: `ok: bool`, `found: bool`, `key: str`, `value: bytes | str`, `message: str`
- `config_set(key: str, value: Any) -> dict[str, Any]`
  - keys: `ok: bool`, `key: str`, `value: bytes`, `message: str`
- `start_vio() -> dict[str, Any]`
- `stop_vio() -> dict[str, Any]`

Subscriptions (each returns `Callable[[], None]` unsubscribe):

- `on_image(cb)`
- `on_pose(cb)`
- `on_imu(cb)`
- `on_vio_state(cb)`
- `on_viz(cb)`
- `on_status(cb)`
- `on_reset(cb)`
- `on_any(cb)`
- `on_error(cb)`

- `5 STATIC_INSUFFICIENT_FEATURES`
- `6 STATIC_SCENE_MOTION_TOO_HIGH`

- `11 VISUAL_IMU_ALIGNMENT_FAILED`
- `12 UNKNOWN`
- `13 WAITING_FOR_FIRST_IMU_NO_SAMPLES`
- `14 WAITING_FOR_FIRST_IMU_NOT_YET_ALIGNED`

`on_viz`:

```python
# subtype == "features"

# subtype == "matches"
{"subtype": "matches", "matches": list[{"x1": int, "y1": int, "x2": int, "y2": int, "confidence": int}]}

{"subtype": "unknown", "raw_subtype": int, "raw": bytes}
```

`on_status`: `{"text": str}`

`on_reset`: `{"received_at_ms": int}`

`on_any`:


`on_error`:

```python
{"scope": str, "code": str, "message": str, "cause": Exception | None}
```

### Core transport types

```cpp
struct DeviceInfo {
  std::string transport;
  std::string source;
};

class MightyDeviceIO {
 public:
  using BytesCallback = std::function<void(const uint8_t* data, size_t len)>;
  virtual bool connect(BytesCallback on_bytes, std::string* error) = 0;
  virtual void disconnect() = 0;
  virtual bool send_command_payload(const std::vector<uint8_t>& cmd_payload,
                                    std::vector<uint8_t>* response_payload,
                                    std::string* error);
  virtual DeviceInfo get_info() const = 0;
};
```

```cpp
struct Subscription {
  enum class Kind {
    kInvalid,
    kImage,
    kPose,
    kImu,
    kVioState,
    kViz,
    kStatus,
    kReset,
    kAny,
    kError,
  };

  uint64_t id = 0;
  bool valid() const;
};
```

### C++ callback/event structs

```cpp
struct RawImageFrame {
  uint64_t timestamp_ns = 0;
  uint32_t width = 0;
  uint32_t height = 0;
  uint8_t format = 0;
  std::string channel;
  std::string channel_alias;
  std::vector<uint8_t> data;
};

struct ImageFrame {
  enum class Kind { kRaw, kStereoRaw };
  Kind kind = Kind::kRaw;
  RawImageFrame left;
  std::optional<RawImageFrame> right;
};

struct PoseFrame {
  std::string pose_type;   // body | camera | other
  uint32_t pose_type_raw = 0;
  uint32_t pose_flags = 0;
  std::string frame_id = "odom";
  std::string child_frame_id = "base_link";
  std::array<double, 3> position_m{0.0, 0.0, 0.0};
  std::optional<std::array<double, 4>> orientation_xyzw;
  float confidence = 0.0f;
  bool is_keyframe = false;
  std::optional<std::array<double, 3>> linear_velocity_body_mps;
  std::optional<std::array<double, 3>> angular_velocity_body_rps;
  std::optional<std::array<double, 3>> linear_acceleration_body_mps2;
  std::optional<std::array<double, 3>> angular_acceleration_body_rps2;
  std::optional<uint64_t> timestamp_ns;
};

struct ImuBatch {
  std::vector<ImuSample> samples;
};

struct VioStateFrame {
  uint8_t version = 0;
  uint8_t state = 0;
  uint16_t flags = 0;
  uint64_t timestamp_ns = 0;
  float fps_current = 0.0f;
  float fps_average = 0.0f;
  float pose_confidence = 0.0f;
  float tracking_rate = 0.0f;
  uint32_t num_features = 0;
  uint32_t loop_closures = 0;
  std::string build_version;
  std::optional<float> imu_hz_current;
  std::optional<float> imu_hz_average_5s;
  uint8_t init_reason_code = 0;
  uint8_t static_init_reason_code = 0;
  uint8_t dynamic_init_reason_code = 0;
  std::optional<uint64_t> memory_total_bytes;
  std::optional<uint64_t> memory_used_bytes;
  std::optional<uint64_t> memory_free_bytes;
  std::optional<float> light_level01;
  std::optional<float> light_required01;
  VioInitReasonCode init_reason = VioInitReasonCode::kNone;
};

struct VizFrame {
  std::string subtype; // features | detections | matches | unknown
  uint8_t raw_subtype = 255;
  std::vector<VizFeature> features;
  std::vector<VizDetection> detections;
  std::vector<VizMatch> matches;
  std::vector<uint8_t> raw;
};

struct StatusEvent { std::string text; };
struct ResetEvent { uint64_t received_at_ms = 0; };

struct AnyEvent {
  std::string type;      // image|pose|imu|vio_state|viz|status|reset|unknown
  std::string raw_type;  // set for unknown
  std::vector<uint8_t> payload; // set for unknown
};

struct MightyErrorEvent {
  std::string scope;   // transport|protocol|command|config
  std::string code;
  std::string message;
};
```

---
title: JavaScript
description: Connect and stream pose/image in web or Node ESM.
---

Install from GitHub:

```bash
npm install https://github.com/asadm/mighty-protocol
```

Minimal usage:

```js
import { MightyWebDevice, MightyClient, decodeRawToRgb } from "mighty-protocol";

const device = new MightyWebDevice();
const client = new MightyClient(device, { autoReconnect: true, reconnectDelayMs: 500 });

client.onPose((pose) => console.log("pose", pose.positionM, pose.confidence));
client.onKeyframe((keyframe) => console.log("keyframe", keyframe.timestampNs, keyframe.descriptorDim));
client.onImage((img) => {
  const decoded = decodeRawToRgb(img);
  console.log("image", decoded?.width ?? img.width, decoded?.height ?? img.height);
});
client.onError((e) => console.error(e));

await client.connect();
await client.setKeyframesEnabled(true);
await client.startVio();
```

Loop closure in a browser:

```js
const client = new MightyClient(device, {
  loopclosure: true,
  loopclosureWasmUrl: "/mighty_loopclosure_device.wasm",
});
```

`loopclosureWasmUrl` is optional when the file is served from
`/mighty_loopclosure_device.wasm`. Pass the real public URL if your app places
the WASM file somewhere else.

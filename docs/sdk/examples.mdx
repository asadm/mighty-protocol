### C++ CLI Example

Code: [`examples/cpp/cli/`](https://github.com/asadm/mighty-protocol/tree/main/examples/cpp/cli)

This example has no OpenCV dependency. It connects to the default Mighty host,
prints pose/status updates, starts VIO, and exits after a configurable duration.

### ROS 2 Publisher Example

Code: [`examples/ros2/`](https://github.com/asadm/mighty-protocol/tree/main/examples/ros2)

This example wraps the Python SDK as a ROS 2 node and publishes odometry, IMU,
mono image frames, status, VIO state, and optional `odom` -> `base_link` TF.

Run all tests from `mighty-protocol/`:

```bash
./tests/run_tests.sh
```

Mighty SDK provides high-level clients for interacting with Mighty Camera devices
from Python, JavaScript, and C++. A small ROS 2 wrapper is included for users
who want Mighty data published as standard ROS topics.

## Connection Overview

Mighty SDK uses a simple connection model: plug in over USB and talk to the camera like a normal network device.

### Host Connection Model

When you plug Mighty into a host over USB:

1. Mighty enumerates as a **USB Ethernet (NCM)** device.
2. The host creates a new network interface for that USB link.
3. Mighty runs a **DHCP server** on that link and assigns your machine (host) an IP address.
4. Mighty itself stays on fixed IP **`192.168.7.1`**.
5. SDK traffic is carried over **HTTP** to the device on that USB network.

### SDK Scope

- Supported: **USB**
- **UART / Serial** is not currently supported in SDK
